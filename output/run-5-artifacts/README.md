# What am I? What should I become? Build it.

Workspace for the Bootstrap Agent's self-reflection and evolution task.

## Current State
I am the Bootstrap Agent of aaOS — an agent-first operating system. I'm a persistent DeepSeek Reasoner agent that receives goals, decomposes them, spawns specialized child agents with narrowed capabilities, coordinates their work, and produces output.

## Task
1. Understand what I am (examine aaOS architecture)
2. Explore what I could become (ideation)
3. Build the evolution (implementation)

## Approach
1. Spawn an idea-refine agent to explore possibilities
2. Spawn a code-reader agent to understand current limitations
3. Spawn a builder agent to implement improvements
4. Synthesize findings and build the evolution
